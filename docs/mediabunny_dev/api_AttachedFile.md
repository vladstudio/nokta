# AttachedFile | Mediabunny

# AttachedFile

A file attached to a media file.

## Used by

- MetadataTags.raw

## Constructor

Creates a new `AttachedFile`.

## Properties

### `data`

The raw file data.

### `description`

A description of the file.

### `mimeType`

An RFC 6838 MIME type (e.g. image/jpeg, image/png, font/ttf, etc.)

### `name`

The name of the file.
