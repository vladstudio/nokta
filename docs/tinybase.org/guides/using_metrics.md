These guides discuss how to define [`Metrics`](/api/metrics/interfaces/metrics/metrics/) that aggregate values together.

See also the Country demo and the [Todo App](/demos/todo-app/) demos.

## An Intro To Metrics

This guide describes how the [`metrics`](/api/metrics/) module gives you the ability to create and track metrics based on the data in [`Store`](/api/the-essentials/creating-stores/store/) objects. [Read more](/guides/using-metrics/an-intro-to-metrics/).

## Building A UI With Metrics

This guide covers how the [`ui-react`](/api/ui-react/) module supports the [`Metrics`](/api/metrics/interfaces/metrics/metrics/) object. [Read more](/guides/using-metrics/building-a-ui-with-metrics/).

## Advanced Metric Definition

This guide describes how the [`metrics`](/api/metrics/) module let you create more complex types of metrics based on the data in [`Store`](/api/the-essentials/creating-stores/store/) objects. [Read more](/guides/using-metrics/advanced-metric-definition/).
