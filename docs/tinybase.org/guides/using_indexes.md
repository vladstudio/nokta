These guides discuss how to define [`Indexes`](/api/indexes/interfaces/indexes/indexes/) that allow fast access to matching [`Row`](/api/store/type-aliases/store/row/) objects.

See also the [Rolling Dice](/demos/rolling-dice/) demos, the [Countries](/demos/countries/) demo, and the [Todo App](/demos/todo-app/) demos.

## An Intro To Indexes

This guide describes how the [`indexes`](/api/indexes/) module gives you the ability to create and track indexes based on the data in [`Store`](/api/the-essentials/creating-stores/store/) objects, and which allow you to look up and display filtered data quickly. [Read more](/guides/using-indexes/an-intro-to-indexes/).

## Building A UI With Indexes

This guide covers how the [`ui-react`](/api/ui-react/) module supports the [`Indexes`](/api/indexes/interfaces/indexes/indexes/) object. [Read more](/guides/using-indexes/building-a-ui-with-indexes/).

## Advanced Index Definition

This guide describes how the [`indexes`](/api/indexes/) module let you create more complex types of indexes based on the data in [`Store`](/api/the-essentials/creating-stores/store/) objects. [Read more](/guides/using-indexes/advanced-index-definition/).
