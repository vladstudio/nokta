These guides discuss how to use [`Checkpoints`](/api/checkpoints/interfaces/checkpoints/checkpoints/) that allow you to build undo and redo functionality.

See also the [Todo App](/demos/todo-app/) demos and the [Drawing](/demos/drawing/) demo.

## An Intro To Checkpoints

This guide describes how the [`checkpoints`](/api/checkpoints/) module gives you the ability to create and track changes to a [`Store`](/api/the-essentials/creating-stores/store/)'s data for the purposes of undo and redo functionality. [Read more](/guides/using-checkpoints/an-intro-to-checkpoints/).

## Building A UI With Checkpoints

This guide covers how the [`ui-react`](/api/ui-react/) module supports the [`Checkpoints`](/api/checkpoints/interfaces/checkpoints/checkpoints/) object. After all, if you have undo functionality in your app, you probably want an undo button! [Read more](/guides/using-checkpoints/building-a-ui-with-checkpoints/).
